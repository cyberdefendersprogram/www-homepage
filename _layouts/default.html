What is the problem?

Cyber attacks are costing companies millions a year. With these threats constantly looming companies cannot afford to be understaffed. The need for specialized talent is increasing in the cybersecurity industry. However, there is an apparent lack of college level education to prepare students for entry level cybersecurity jobs. Many students are not aware of the variety of careers available to them in cybersecurity.

The Solution

The Cyber Defenders mission is to help close the cybersecurity workforce gap by training professionals seeking new opportunities in cybersecurity or related fields.The Cyber Defenders program finds untapped potential, provides them job ready cybersecurity skills without the burden of years of debt, and helps them earn incomes that lift their local communities.

Cyber Defenders Fosters Specialized Talent

Programs that develop technical and professional skills needed for entry level security positions

  • Skill Development
  • Research

  • Activities & Networking

Network Security

From Introducing the fundamentals of networking to writing security programs with python

Penetration Testing & Red Teaming

Hands-on learning with Kali Linux, Metasploit and network analysis tools. From there students get additional practice through fun and engaging Capture the Flag events and other War Games

Web Application Testing

Utilizing the OWASP Top 10 in order to find bugs within bug bounty programs using Burp Suite

Machine Learning & AI

Learning the fundamentals of Machine Learning and AI. Understanding how they apply to cybersecurity especially analyzing network, malware and other threat data

Reverse Engineering

Specifically focusing on the basics of malware reverse engineering. Understanding the tools and concepts that go with it (Need to Edit)

Python Programming

Python scripting for use in building network analysis and other security tools

Incident Research

Getting familiar with the current state of cybersecurity. Picking a part previous attacks to understand how attacks can happen.

Researching Tools

Using Google Scholar, Github Repositories, News Articles, White Papers and other sources. These combined are great for brainstorming projects, finding the right tools and learning new concepts.

Self-Learning

Proficient cyber security professionals have the ability to constantly learn new concepts even when there is not much guidance. Cyber Defenders puts emphasis on taking control of learning and persisting to understand foreign ideas.
